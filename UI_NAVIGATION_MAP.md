## 🚦 Navigation Rules

### **From Any View:**
✅ Can reach Glass Showcase
✅ Can reach any other view
✅ Maximum 1 click away
✅ Clear button labels
✅ Consistent positions

### **To Glass Showcase:**
- Always labeled "Main View" or "Go to Glass UI"
- Primary color (stands out)
- Star/Sparkle icon
- Clear visual hierarchy

### **Between Alternative Views:**
- Direct navigation available
- Multiple paths possible
- No forced routing
- User choice respected
