# ⚡ FlowForge AI - Quick Reference

## 🎯 Default Interface: Glass Showcase

**Login → Glass Showcase UI loads automatically**

---

## 🚀 Quick Actions

| Action | How To |
|--------|--------|
| **View Workspaces** | Scroll top-left glass panel |
| **Create Workspace** | Click "Create New Workspace" button |
| **Use AI** | Click AI Command Center panel |
| **See Analytics** | Click Reports & Analytics panel |
| **Change Settings** | Click gear icon |
| **Switch View** | Use bottom toggle buttons |

---

## 🎨 Available Views

### **1. Glass Showcase** ⭐ DEFAULT
- **Beautiful UI** with animations
- **All features** included
- **Access**: Default on login

### **2. Integration Canvas Glass**
- **Workflow builder** with glass effects
- **Drag & drop** integration cards
- **Access**: Bottom → "Integration Canvas"

### **3. Classic Canvas**
- **Workflow builder** solid colors
- **Cream background** professional look
- **Access**: Bottom → "Classic Canvas"

### **4. Classic Dashboard**
- **Traditional UI** business-ready
- **Full features** no animations
- **Access**: Bottom → "Dashboard"

---

## 🔄 View Switching

### **From Glass Showcase:**
Bottom buttons → [Integration Canvas] [Classic Canvas] [Dashboard]

### **From Any Other View:**
Look for → [⭐ Main View] or [✨ Go to Glass UI]

## 💡 Pro Tips

### **Performance**
- Works best on modern devices
- GPU required for smooth animations
- Toggle to Classic for slower devices

### **Customization**
- Settings → Adjust blur, particles, animations
- Pick your preferred gradient colors
- Toggle features on/off

### **Navigation**
- All panels are collapsible
- Use toggle buttons to switch views
- Keyboard shortcuts in Settings

---

## 🎨 Color Guide

| Element | Color |
|---------|-------|
| **Background** | Purple → Pink → Blue → Cyan gradient |
| **Glass Panels** | White 10-20% opacity + blur |
| **Primary Buttons** | Purple/Pink gradient |
| **Secondary Buttons** | Blue/Cyan gradient |
| **Text** | White on dark, Dark on light |

## 🆘 Common Questions

**Q: How do I get back to Glass Showcase?**
A: Click [⭐ Main View] or [✨ Go to Glass UI] button

**Q: Can I disable animations?**
A: Yes! Settings → Animations → Off

**Q: Which view is fastest?**
A: Classic Dashboard (no animations)

**Q: Which view is most beautiful?**
A: Glass Showcase (default)

**Q: Can I customize colors?**
A: Yes! Settings → Appearance → Gradient
